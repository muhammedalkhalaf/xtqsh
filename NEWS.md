# xtqsh 1.0.0

* Initial release.

* Implements the Galvao, Juhl, Montes-Rojas, and Olmo (2017) test for slope
  homogeneity in panel quantile regression models.

* Features:
  - Joint slope homogeneity test (S-hat and D-hat statistics)
  - Marginal tests for individual coefficients
  - Minimum distance quantile regression estimator
  - Hall-Sheather and Bofinger bandwidth selection
  - Optional HAC variance estimation
  - Comparison with OLS (mean regression) test
